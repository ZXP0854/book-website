# 图书推荐网站

一个使用浮动布局和图文混排技术的图书推荐页面。

## 访问方式

部署到GitHub Pages后访问：
- https://yourusername.github.io/book-website/实验2.html

## 页面说明

- `实验2.html` - 主页面（图书推荐）
- `copyright.html` - 版权声明页面
- `头像.png` - 开发者头像
- `图片素材/` - 图片文件夹

## 部署到GitHub Pages

1. 将此文件夹上传到GitHub仓库
2. 进入仓库 Settings → Pages
3. Source 选择 `main` branch
4. 等待几分钟后即可访问

## 本地预览

```bash
python -m http.server 8080
# 然后访问 http://localhost:8080/实验2.html
```
